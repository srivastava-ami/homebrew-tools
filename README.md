# homebrew-tools

Homebrew tap for [srivastava-ami](https://github.com/srivastava-ami) tools.

## Install

```bash
brew tap srivastava-ami/tools
```

Then install any tool:

| Tool | Description |
|---|---|
| `brew install coderev` | Deterministic, polyglot code-standards enforcement |

## Add a new formula

Each tool's release workflow pushes its `Formula/<name>.rb` here automatically.
